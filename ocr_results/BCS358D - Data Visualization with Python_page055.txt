6. The resulting plot should show a histogram of the randomly generated data with 20 bins,
displaying the distribution of values and their frequencies. The transparency (alpha) is set

to 0.7, and the edges of the bins are colored black for better visibility.

PROGRAM
import matplotlib.pyplot as plt
import numpy as np

# Generate random data for the histogram
data = np.random.randn(1000) # Sample 1000 data points from a normal
distribution

# Create a histogram plot
plt.hist (data, bins=20, edgecolor='black', alpha=0.7)

# Add labels and a title

plt.xlabel ('Values')

plt.ylabel ('Frequency')
plt.title('Histogram Plot Example')

# Show the plot
plt.show()
