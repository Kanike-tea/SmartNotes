4.4 DATA PROCESSING INSTRUCTIONS

The data processing instructions manipulate data within registers. They include move
instructions, arithmetic instructions, shifts, logical instructions, comparison instructions,
and multiply instructions. The Thumb data processing instructions are a subset of the ARM
data processing instructions.
