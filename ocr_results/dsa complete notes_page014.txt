Complexity of the bubble sort algorithm is O(n?)
e The time for sorting is measured in terms of the number of comparisons. In the bubble sort
there are n-1 comparison in the first pass which places the largest element in the last position.
e There are n-2 comparison in the second pass which places the second largest element in the
next to last position and so on, therefore
C(n)= (n-1) +(n-2) + 2+ 1= n(n-1)/2= n?/2 = O(n)
