Example:

MOVS r0,r1, LSL #1 ; rl register content left shifted by one position and copied to r0 and
CPSR register get updated because of suffix S.

PRE cpsr = nzcvqiFt_USER, r0 = 0x00000000, rl = 0x80000004
POST epsr = nzCvqiFt_USER, r0 = 0x00000008, r1 = 0x80000004

This example of a MOVS instruction shifts register r1 left by one bit. This multiplies register
rl by a value 2. C flag is updated in the cpsr because the S suffix is present in the instruction

mnemonic.
